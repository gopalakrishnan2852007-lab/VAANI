# VAANI — Voice Assisted Accessible Native Interface

> "Speak Your Language. Access Your World."

## What is VAANI?

VAANI is an AI-powered multilingual voice assistant that helps Indian citizens access government services in their native language (Tamil, Hindi, Telugu, Malayalam).

## Features

- 🎤 Voice input in Tamil, Hindi, Telugu, Malayalam
- 🤖 AI-powered responses using Google Gemini
- 🔊 Text-to-Speech output in local language
- 🏛️ Government services: Ration Card, Hospital, Bus, Aadhaar, PAN, Schemes
- 📱 Works on any smartphone browser
- ⚡ No app installation needed

## Setup

1. Get free Gemini API key from: https://aistudio.google.com
2. Open `index.html`
3. Replace `YOUR_GEMINI_API_KEY` with your key
4. Deploy on Vercel

## Deploy on Vercel

```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
vercel

# Follow prompts → Get live URL
```

## Tech Stack

- HTML5 + CSS3 + JavaScript
- Web Speech API (voice input/output)
- Google Gemini API (AI responses)
- Vercel (hosting)
